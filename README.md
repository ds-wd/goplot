# `goplot` - terminal based stream plotting

![Sample plots](https://cloud.githubusercontent.com/assets/234997/23121604/ebbe9cb4-f715-11e6-8541-6881d19aadd1.png)

## Install:

```
$ go get github.com/lebinh/goplot
```
